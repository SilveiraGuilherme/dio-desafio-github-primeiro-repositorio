Link para download do Git: https://git-scm.com/downloads
O Git Bash é um terminal extendido para otimizar o uso do Git.

Passos para baixar arquivos do GitHub para o PC:
- Copiar o link HTTPS no GitHub
- No Git Bash, digitar git clone <link>

Passos para enviar arquivos do PC para o GitHub:
- git add . ou git add -A (Move os arquivos para unstage)
- git commit -m "comentário do commit" (Comita os arquivos)
- git push origin main (Envia todos os commits locais pra nuvem)

*O comando git status mostra a situação em que o arquivo se encontra.
